Микросервис на Go, который создает графики из json файлов.
Он:
Получает один большой список пингов и разделяет их по сайтам (google.com, ya.ru и т.д.).
Генерирует графики задержки (latency) через QuickChart API.
Отдает ZIP-архив: Упаковывает все картинки в один файл charts.zip.
Как это работает:
Запрос: Отправь POST на /api/chart/sync с массивом данных:

JSON
[{ "date": "2026-03-29T13:47Z", 
"domain": "google.com", "
latency": 25 }]
Ответ: архив charts.zip с готовыми PNG для сайта.
Запуск:
Bash
go run main.go
Сервис поднимется на порту :8080
